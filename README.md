**Student Performance Analysis Using Python**

This project focuses on analyzing student performance data using Python libraries such as Pandas, Matplotlib, and Seaborn. The main objective of the project is to understand how different factors like gender, parental education, ethnic groups, and test preparation affect students’ academic performance in Mathematics, Reading, and Writing.

During the project, the dataset was cleaned and processed by handling missing values, removing unnecessary spaces, checking duplicate records, and correcting data types to ensure accurate analysis and visualization.

**Key Insights from the Analysis:**

The distribution of students across different ethnic groups was analyzed using pie charts and count plots.

From the analysis, it was observed that students are comparatively weaker in Mathematics than in Reading and Writing.

The education level of parents has a positive impact on students’ academic performance.

The dataset analysis also showed that the number of female students is higher than the number of male students.
